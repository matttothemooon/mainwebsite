# mainwebsite

Personal landing page for mattothemoon.xyz.

## About

A single-page personal site for mattothemoon.xyz — terminal-styled landing page with links, live Discord/Spotify status, and mod experience.

## Structure

- `index.html` — the site
- `styles.css` — styling
- `site.js` — Discord/Spotify status, interactivity
- `vercel.json` — clean URLs, trailing slash config, and redirects (e.g. `/discord`, `/twitch`, `/twitter`, `/youtube`)
- `robots.txt` / `sitemap.xml` — SEO
